## ml5.js
- **Website:** https://ml5js.org
- **Tags:** open-source, javascript, p5js, education
- **License:** MIT
- **Overview:** Friendly machine learning library that brings models like StyleGAN and PoseNet into p5.js sketches with minimal code.
